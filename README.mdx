
# Настройка кастомного сервера

Этот проект представляет собой кастомный сервер, созданный на **чистом Node.js** и **PostgreSQL** (без использования таких фреймворков, как Express). Ниже приведено пошаговое руководство по его запуску.

## Начало работы

### 1. Установка зависимостей
Для начала установите необходимые npm-пакеты:
```
npm install
```

### 2. Настройка окружения
Перед запуском сервера необходимо создать файл \`.env\` в корневом каталоге проекта. В этом файле укажите следующие переменные окружения для подключения к базе данных PostgreSQL:

```
DB_USER=ваш_логин_от_базы_данных
DB_PASSWORD=ваш_пароль_от_базы_данных
DB_NAME=ваше_название_базы_данных
DB_HOST=localhost
DB_PORT=5432
```

Убедитесь, что заменили \`ваш_логин_от_базы_данных\`, \`ваш_пароль_от_базы_данных\`, \`ваше_название_базы_данных\` на актуальные данные для подключения к PostgreSQL.

### 3. Запуск сервера
После настройки окружения и установки зависимостей, сервер можно запустить командой:
```
npm run start
```

### Используемые технологии
- **Node.js** (без фреймворков)
- **PostgreSQL** (в качестве базы данных)

Этот сервер представляет собой кастомное решение, разработанное с нуля на чистом Node.js и использующее PostgreSQL для работы с базой данных, предоставляя гибкость в управлении серверной логикой без использования внешних фреймворков.
